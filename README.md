# MyBooks-App
OOSE assignment 1

## how to run the app
# run backend
1. 'npm install' in MyBooks-App folder
2. 'node server.js'
# run frontend
1. open another terminal while keep the backend running, then run frontend
2. 'cd frontend' to navigate to frontend folder
3. 'npm i'
4. 'npm run start'
5. app running on http://localhost:3000/!
